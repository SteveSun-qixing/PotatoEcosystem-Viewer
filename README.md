# 👁️ Chips-Viewer

> 薯片生态 - 卡片查看器

## 📖 简介

Chips-Viewer 是薯片生态系统中的只读卡片查看应用，用于查看和展示卡片内容，无需编辑功能。

## ✨ 特性

- 📄 支持多种卡片格式查看
- 🎨 主题切换支持
- 📱 响应式布局
- ⚡ 快速加载和渲染
- 🔗 支持卡片间链接跳转

## 🚀 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```

## 📦 依赖

- `@chips/sdk` - 薯片 SDK
- `@chips/foundation` - 公共基础层
- `@chips/components` - 组件库
